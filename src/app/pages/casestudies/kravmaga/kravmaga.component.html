KravMaga Center

KravMaga center is a gym based in Luxembourg, dedicated to the practice and philosophy of Krav Maga: the Self-Defense, Fitness and Fighting.

Project Overview

The project consisted of the rebranding and development of their website. Development included a registration system, a blog, an event management system, and the integration of their existing online shop and academy platform.

Discovery

The client needed a complete website rebrand and development, with additional new features.

First, the website has to showcase the identity of the center, present enough (and clear) information about it (copywriting), and be fully responsive on all devices.

The user experience had to be on point, guiding the users along their visit with well structured content and engaging call to actions, with the goal to increase conversions.

Additionally, a set of features had to be implemented: Event Management System, Program Registrations, Blog, Integration of Programs Schedule from another platform, Internalization (English and French).

Implementation

The old website was already running on php, the web host supporting only php, I carried on with it, and used the Laravel framework to develop the new website.

Frontend was carefully crafted using mostly plain HTML/CSS and vanilla JavaScript, along with Bootstrapâ€”to make layouts easily responsive. I used Laravel templating engine for serving the pages, and for the routing.

Other features required some backend heavy lifting. The event management system allows the administrator of the website to schedule upcoming events, and to post feedback from the event. The Blog was implemented from scratch, but was fairly simple for their needs. Internalization was straight forward in Laravel. Through the admin panel, events, blog posts, registrations, and user inqueries can all be managed. Finally, their Programs Schedule was integrated through iframe tag provided by their Scheduling platform.

Results

After testing thouroughly the functionality of all the features, the website was launched.
